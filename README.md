# Rust Terminal To-Do List

Mini projeto de To-Do List em Rust executado no terminal.  
Permite adicionar, listar, marcar como concluída e remover tarefas usando uma interface de linha de comando (CLI).  
O objetivo é praticar conceitos fundamentais do Rust como `struct`, `Vec`, `loop`, `match`, entrada do usuário (`stdin`) e manipulação de dados.

---

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Marcar tarefa como concluída
- Remover tarefa
- Exibir número atual de tarefas
- Interface interativa no terminal

---

## Como executar

### 1. Instalar Rust (caso não tenha)

```bash
curl https://sh.rustup.rs -sSf | sh
```

Reinicie o terminal e verifique:

```bash
rustc --version
```

---

### 2. Clonar o repositório

```bash
git clone https://github.com/seu-usuario/todo_terminal.git
cd todo_terminal
```

---

### 3. Executar o projeto

```bash
cargo run
```


## Requisitos

- Rust 1.70+
- Cargo (incluído com Rust)
